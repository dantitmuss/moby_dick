About This Project

The Moby-Dick Bible is an experiment in reverent absurdity — a way of treating Herman Melville’s epic like a sacred text, full of strange truths and slippery meanings.

It begins by writing a new verse in Melville’s voice, using your idea or emotion as inspiration. Then, it searches the book to find real passages that resonate semantically.

The result is part scripture, part hallucination, and part literary concordance.

How it Works

The Moby-Dick Bible uses a new way of searching which first imagines what a user is searching for, then gives them the best matches according to that imagined verse. It does this by first generating the verse with AI, then matching it to an embedding using FAISS.

  • Your input is used to generate a new verse in Melville’s style using an AI model.
  • That verse is embedded and compared to all verses in Moby-Dick.
  • The closest matches are returned — semantically, not literally.
Built using Python, Flask, Tailwind, OpenAI, and a stubborn love for strange books. See for yourself on Github.
